# 🎉 Party Games App

A fun React-based party games application that's iOS compatible!

## ✨ Features

- 🔗 **Word Chain** - Build word chains in multiplayer mode
- 🧠 **Trivia Challenge** - Test your knowledge with friends
- 🤔 **Social Guessing** - Collaborative guessing game

## 🚀 Quick Start

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## 📱 iOS Deployment Options

### Option 1: PWA (Progressive Web App) ⭐ Easiest
The app is already configured as a PWA! Just host it and users can:
- Add to home screen on iPhone
- Run fullscreen like a native app
- Works offline after first load

### Option 2: React Native Wrapper
Convert to native iOS app using tools like:
- **Capacitor** - Web-to-native bridge
- **React Native WebView** - Wrap web app in native container
